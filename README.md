# Gestion d'établissement scolaire

Application web de gestion d’établissement scolaire permettant de centraliser la gestion des stagiaires, groupes, modules, notes et absences.
Le projet intègre également un système multi-rôles avec des espaces dédiés pour l’administrateur, le formateur et le stagiaire.

## 🛠️ Technologies utilisées
- PHP / Laravel
- MySQL
- React.js
- JavaScript
- HTML5 / CSS3
- Axios
- React Router
- Framer Motion
- Chart.js / Recharts
- jsPDF / XLSX

## ⚙️ Installation et lancement
```bash
# Frontend
cd frontendpfe
npm install
npm start

# Backend
cd C:\xampp\htdocs\PFE
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
